École Escale

Pour que toute l'équipe voie les mêmes données en temps réel,
connectez une base de données Firebase gratuite.

1
Ouvrez console.firebase.google.com et connectez-vous avec un compte Google
2
Cliquez Créer un projet → donnez un nom (ex : escale-rh) → désactivez Google Analytics → Créer
3
Menu gauche : Build → Realtime Database → Create Database → région europe-west1 → Start in test mode → Enable
4
Dans Realtime Database → Rules, remplacez les règles par celles-ci et cliquez Publish :
Règles Firebase à coller { "rules": { ".read": true, ".write": true } }
5
Revenez sur Realtime Database → Data, copiez l'URL affichée (ex : https://escale-rh-default-rtdb.europe-west1.firebasedatabase.app)
6
Collez cette URL ci-dessous. Chaque collaborateur devra faire cette étape une seule fois sur son navigateur.

L'URL est enregistrée uniquement dans ce navigateur. Les données restent dans votre base Firebase.
